 Task Manager Dashboard

 Description

The Task Manager Dashboard is a web-based application developed using HTML, CSS, and JavaScript to manage daily tasks efficiently. It provides a simple interface for users to add, delete, and track tasks with real-time updates.

 Features

- User login (Admin / Member role selection)
- Add new tasks
- Delete tasks
- Mark tasks as completed or pending
- Dashboard showing total, completed, and pending tasks
- Data stored using browser localStorage

 Technologies Used

- HTML
- CSS
- JavaScript

 How It Works

1. User logs in as Admin or Member
2. Admin can add tasks
3. Users can mark tasks as completed or pending
4. Tasks can be deleted anytime
5. Dashboard updates automatically

How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click "index.html"
4. Use the application in your browser

Project Structure

- "index.html" – Structure of the application
- "style.css" – Styling and layout
- "script.js" – Application logic

 Future Enhancements

- User authentication with password
- Backend integration (Node.js, MongoDB)
- Notifications and reminders
- Improved UI design

 Conclusion

The Task Manager Dashboard is a simple and effective web application for organizing and tracking daily tasks. It demonstrates core web development concepts using HTML, CSS, and JavaScript, including dynamic updates and local data storage. The project provides a user-friendly interface and can be further enhanced with backend integration and advanced features.
